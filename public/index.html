File Metadata Microservice

User Stories:

  1. User Story: I can submit a FormData object that includes a file upload.
  2. User Story: When I submit something, I will receive the file size in bytes within the JSON response.

TRY IT:

The following upload form will post to the following url, display the file size in JSON format, and then delete the original file from the server. https://file-metadata-alexanderlamm.c9users.io/api/fileanalyse
